[[update]]
=== Updating MIDS

Updates for MIDS will be automatically detected each time the tool is started.
If updates are detected, a notification will pop-up at the bottom right corner.
Also an update icon will appear in the bottom right corner.

image::{imgsdir}/update-available.png[]

Click on the dialog or click on the update-icon, it will start a standard update procedure.
Follow the steps and restart the tool to complete updating.

Alternatively, select the menu:Help[Check for Updates] menu.
Note that a restart may be required to detect updates released while the tool is running.
