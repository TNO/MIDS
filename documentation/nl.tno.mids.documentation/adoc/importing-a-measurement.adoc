== Importing a measurement

MIDS provides <<cmi,Constructive Model Inference>> to automatically infer models from execution traces.
CMI works on execution traces, in the form of Timed Message Sequence Charts (TMSCs). 
The details of how to produce a TMSC from your company's proprietary logging format, are beyond the scope of this documentation.
